# Инструкция по реакции на новость

Цель — определить, можно ли спекулятивно заработать на конкретной новости о компании. Результат — JSON trade_signal с условиями входа и выхода.

Данные (цена, новость, orderbook, предыдущие сигналы) уже собраны скриптом `prepare_news_context.py` и переданы тебе в промпте. Твоя задача — принять решение.

## Классификация новости

**Тип новости:**

| Тип | Примеры | Типичное влияние на цену |
|-----|---------|--------------------------|
| `results` | Квартальный/годовой отчёт, предварительные результаты | Сильное, если расхождение с ожиданиями |
| `dividends` | Объявление дивидендов, изменение payout | Среднее-сильное |
| `regulation` | Санкции, налоги, блокировки ФНС, изменение тарифов | Сильное |
| `corporate` | M&A, buyback, допэмиссия, делистинг, смена менеджмента | Сильное |
| `macro` | Ставка ЦБ, курс, инфляция (влияет на компанию) | Среднее |
| `noise` | Мнения аналитиков, пересказ старых новостей, прогнозы | Слабое → **skip** |

**Проверка свежести:** Если новость старше 2 торговых дней → **skip** (рынок уже отработал). Исключение: вышла в выходные/после закрытия.

## Проверка «уже в цене»

**Матрица решений по price_move и volume_ratio (данные в промпте):**

| price_move (абс.) | volume_ratio | Вердикт |
|--------------------|-------------|---------|
| < 2% | любой | Рынок **не отреагировал** → окно открыто |
| 2–5% | < 2× | **Частично в цене**, но есть пространство |
| 2–5% | > 2× | **В цене** → ищем перепроданность (негатив) или skip (позитив) |
| > 5% | любой | **Сильная реакция** → возможна перепроданность / перекупленность |

**КЛАСТЕР:** Если в промпте есть секция «Кластер новостей» — используй накопленный price_move. Если > 2% при позитиве → позитив УЖЕ В ЦЕНЕ → skip.

**Сценарии для заработка:**

| Сценарий | Когда | Логика |
|----------|-------|--------|
| **long-positive** | Позитивная новость + price_move < 2% + НЕТ кластера | Покупка до реакции рынка |
| **long-oversold** | Негативная новость + price_move > 5% вниз + фундаментал НЕ сломан | Покупка на панике, рынок переоценил негатив |
| **short-negative** | Негативная новость + price_move < 2% + фундаментал сломан или ухудшен | Шорт до реакции рынка |
| **short-overbought** | Позитивная новость + price_move > 5% вверх + upside < 10% | Шорт на перекупленности, рынок переоценил позитив |

**Для short-сценариев дополнительно:**
- Проверить: бумага доступна для шорта у брокера (ликвидные бумаги с ADV > 200M обычно доступны)
- Stop-loss для шорта: +10% от entry (или максимум дня при short-overbought)
- Target для short-negative: `fair_value` если он ниже текущей цены, иначе −10–15% от entry
- Target для short-overbought: `pre_news_price`

Если ни один сценарий не подходит → **skip**.

## Оценка фундаментала

**Ключевой вопрос: новость МЕНЯЕТ фундаментальный тезис или нет?**

| Меняет тезис | Примеры | Действие |
|-------------|---------|----------|
| **Да, ломает** | Делистинг, дефолт, потеря лицензии, SDN-санкции | **Skip** |
| **Да, усиливает** | Рекордная прибыль при bearish-consensus, неожиданные дивиденды | Торгуем с высокой уверенностью |
| **Нет, временная** | Блокировка счетов, разовый штраф, слухи | Торгуем, если цена перереагировала |
| **Нет, шум** | Переупаковка известной информации | **Skip** |

**Для long-oversold дополнительно:**
- `key_risks` — если новость подтверждает известный риск, падение может продолжиться
- `Net Debt/EBITDA > 3x` + новость про долг → осторожно
- Есть ли ближайший катализатор (отчёт, дивиденды, ЦБ), который развернёт sentiment?

## Расчёт trade setup

**Entry:**

| Сценарий | Entry |
|----------|-------|
| long-positive | По рынку (текущая цена) |
| long-oversold | Текущая цена или лимитка на уровень поддержки |
| short-negative | По рынку (текущая цена) |
| short-overbought | Текущая цена или лимитка на уровень сопротивления |

**Target:**

| Сценарий | Target |
|----------|--------|
| long-positive | При сильном однозначном катализаторе — `fair_value` (полный upside). При слабом/неоднозначном катализаторе — +5–10% от entry (реалистичный short-term target) |
| long-oversold | `pre_news_price` (консервативно) или `fair_value` (агрессивно) |
| short-negative | `fair_value` если ниже entry, иначе −10–15% от entry |
| short-overbought | `pre_news_price` |

**Stop-loss:**

| Тип новости | Stop-loss | Логика |
|-------------|-----------|--------|
| results | −10% от entry | Волатильность выше на отчётах |
| dividends | −5% от entry | Узкий стоп, идея конкретная |
| regulation / corporate | −10% от entry | Регуляторный риск непредсказуем |
| long-oversold | Минимум дня паники (low) | Пробьёт дно — падение продолжится |
| short-negative | +10% от entry | Если рынок не согласен — закрываем |
| short-overbought | Максимум дня (high) | Пробьёт хай — рост продолжится |

**Приоритет стоп-лосс правил:** Сценарий-специфичный стоп имеет приоритет над типом новости. Пример: `long-oversold` на новости типа `regulation` → стоп = минимум дня паники (не −10% от entry). Тип новости определяет стоп только когда для сценария нет специфичного правила.

**Risk/reward:**

```
# Long:
expected_return = (target - entry) / entry × 100%
risk = (entry - stop_loss) / entry × 100%
risk_reward = expected_return / risk

# Short:
expected_return = (entry - target) / entry × 100%
risk = (stop_loss - entry) / entry × 100%
risk_reward = expected_return / risk
```

**Минимальный risk_reward: 2.0.** Если < 2.0 → **skip**.

**Time limit:**

| Сценарий | Таймфрейм |
|----------|-----------|
| long-positive (отчёт/дивиденды) | 5–10 торговых дней |
| long-oversold (паника) | 10–20 торговых дней |
| long-positive (макро, ставка ЦБ) | до следующего заседания ЦБ |
| short-negative | 5–10 торговых дней |
| short-overbought | 3–5 торговых дней |

**Размер позиции:**

| Confidence | Позиция | Когда |
|------------|---------|-------|
| high | full (до 5% портфеля) | risk_reward > 3, новость однозначная, ликвидность высокая |
| medium | half (до 3% портфеля) | risk_reward 2–3, есть неопределённость |
| low | **skip** | risk_reward < 2 или слишком много неизвестных |

## Решение

| Сигнал | Условия |
|--------|---------|
| **buy** | risk_reward ≥ 2.0 + катализатор понятен + ликвидность ок + тезис не сломан |
| **sell** | risk_reward ≥ 2.0 + негатив однозначен + бумага шортится + ADV > 200M |
| **skip** | Всё остальное |

## Предыдущие сигналы

Если в промпте есть секция «Предыдущие сигналы» — проверь: если по той же теме уже были skip-ы, новая новость той же темы тоже **skip** (не переоценивать ту же информацию).

## Антипаттерны

| Ошибка | Как избежать |
|--------|--------------|
| Покупка на негативе без проверки фундаментала | Тезис сломан → skip |
| Покупка позитива после реакции цены | price_move > 5% + volume > 2× ADV → skip |
| Игнорирование risk/reward | r/r < 2 → skip |
| Торговля без time limit | Всегда ставь time_limit |
| Реакция на шум | strength: low → skip |
| Торговля неликвидом | ADV < 50 → skip |
