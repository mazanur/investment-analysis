# ВТБ (VTBR)

## Бизнес-модель

Второй по активам банк России (~20% активов банковской системы). Системообразующая кредитная организация (СЗКО), контрольный пакет принадлежит государству (~75% после конвертации префов — в процессе завершения, ожидается до мая 2026).

**Основные направления:**
- **Корпоративный банкинг** (~55% процентных доходов): кредитование крупного и среднего бизнеса, торговое финансирование, проектное финансирование
- **Розничный банкинг** (~30%): ипотека, потребительские кредиты, депозиты. Кредитный портфель: ~7,3 трлн рублей
- **Инвестиционный банкинг**: ВТБ Капитал — одна из крупнейших площадок для DCM/ECM
- **Дочерние банки**: интеграция Почта Банка, «Открытие» (завершено)

**Масштаб:**
- Кредитный портфель: ~24,4 трлн рублей (2-е место после Сбера)
- Депозитная база: ~27 трлн рублей
- Чистая прибыль FY 2025: **500 млрд рублей** (факт), прогноз 2026: 600–650 млрд

**Драйверы роста:**
1. Снижение ставки ЦБ 15,5% → 13% к концу 2026 → рост NIM и ROE (~+0,44 п.п. ROE за каждые -1 п.п. ставки)
2. Дивиденды за 2025 год (25–50% прибыли), ожидаются июль 2026
3. Конвертация привилегированных акций → упрощение структуры капитала, улучшение Н20.0
4. Рост прибыли: 500 млрд (2025) → 600–650 млрд (2026E) = +20–30% YoY

## Конкурентные преимущества

- **Масштаб**: 2-е место по активам в РФ, доступ к крупнейшим корпоративным клиентам
- **Государственная поддержка**: фактический гарант от банкротства, регулярная докапитализация
- **Широкая сеть**: присоединение «Открытия» и Почта Банка дало доступ к розничной клиентской базе
- **Инфраструктурная роль**: участие в крупнейших госпроектах, торговом финансировании

## Риски

### Макро-риски
- Медленное снижение ставки ЦБ сдерживает рост NIM (NIM 11M 2025: ~1,3% — крайне низко)
- Рост просроченной задолженности: NPL 3,8% (снизился с 4,3%), стоимость риска 1,0%
- Провизии резко выросли: +150,4% YoY за 11M 2025 — кредитное качество ухудшается

### Операционные риски
- **Дефицит капитала**: CET1 = 6,1% — критически низкий уровень. Н20.0 = 9,6% при минимуме 9,25%
- **Конвертация префов 2026**: привилегированные акции на 521 млрд рублей планируется конвертировать до мая 2026. Параметры не раскрыты. По базовому сценарию разводнение ~35% акций (недискриминационный коэффициент)
- **Cost/Income растёт**: 47,9% за 11M 2025 vs 44,5% ранее — умеренное ухудшение
- **NIM компрессия**: 1,3% (11M 2025) — крайне низко для банка, восстановление идёт медленно

### Регуляторные риски
- Санкции: ВТБ под полными блокирующими санкциями США/ЕС, отключён от SWIFT
- Ужесточение надбавок для СЗКО до 2,0 п.п. с 2026 года — дополнительное давление на капитал
- Антициклическая надбавка 0,5 п.п. с июля 2025

### Корпоративные риски
- Непредсказуемая дивидендная политика: 0 дивидендов за 2019–2023, затем 50% за 2024, 25–50% за 2025
- История допэмиссий и размытия (2015, Q3'25 +1,25 млрд акций, конвертация 2026)
- Менеджмент без KPI на капитализацию

## Мой тезис

ВТБ завершил 2025 год с прибылью 500 млрд рублей (-6,5% YoY), что подтверждает устойчивость в сложной ставочной среде. ROE восстановился до 17,4% по итогам 11М (ранее 14,5%) — главная позитивная новость. NIM в ноябре 2025 ускорился до 2,1%, что показывает начало нормализации. NPL снизился с 4,3% до 3,8% — менее критично, чем опасались.

Риски сохраняются: (1) CET1 6,1% — критически низкий, буфер над минимумом только 0,35 п.п.; (2) конвертация привилегированных акций до мая 2026 — параметры окончательно не раскрыты, Минфин и ВТБ «близки к договорённости»; (3) провизии выросли +150% YoY — качество кредитного портфеля вызывает вопросы; (4) история систематического размытия миноритариев.

Снижение ставки ЦБ (15,5% в феврале 2026) и прогноз 13–14% на конец 2026 — структурный позитив. Каждые -1 п.п. ставки добавляет ~+0,44 п.п. к ROE. Прогноз прибыли на 2026: 600–650 млрд = +20–30% к 2025. При реализации этого сценария forward P/E ~1,7x — дёшево.

**Position: watch** — позитивные изменения есть (ROE 17,4%, NIM восстанавливается, прогноз 2026 позитивный), но конвертация префов остаётся ключевым корпоративным риском. Переход в buy — после прояснения параметров конвертации и подтверждения недискриминационного коэффициента.

**Катализаторы:**
1. Объявление параметров конвертации префов (недискриминационный коэффициент) — ожидается до мая 2026
2. Публикация FY 2025 IFRS (25 февраля 2026) — возможное подтверждение прибыли 500 млрд
3. Дивидендное решение за 2025 год (50% → ~17 ₽/акция → ~20% доходность)
4. Снижение ставки ЦБ → рост ROE → переоценка акций

**Антикатализаторы:**
1. Конвертация по номиналу (50 руб) → удвоение акций → обвал EPS/DPS
2. Ухудшение достаточности капитала ниже минимума → новая допэмиссия
3. Отказ от дивидендов за 2025 год из-за нехватки капитала

## Целевая цена

### Метод: Justified P/BV (основной для банков)

**Обновлённые параметры (ставка ЦБ 15,5%, февраль 2026):**
- Кол-во акций после допэмиссии Q3'25: ~7 870 млн шт.
- BV per share = 2 900 млрд / 7 870 млн = **~368 ₽**
- ROE нормализованный: **17,5%** (факт 11М = 17,4%, прогноз 2026 улучшается)
- r = ставка ЦБ + ERP = 15,5% + 5% = **20,5%**
- g = 5% (зрелый банк, рост ≈ инфляция)
- Спред r − g = 15,5% ✓ (> 5%)

```
Justified P/BV = ROE / (r − g) = 0.175 / (0.205 − 0.05) = 0.175 / 0.155 = 1.13x
```

**Поправка за качество:**
- NIM ниже сектора: **-10%**
- Лидер сектора по масштабу: **+10%**
- NPL выше среднего (3,8%): **-5%**
- CET1 критический: **-10%**
- Итого: -15% → 0.85x

```
Целевой P/BV = 1.13 × 0.85 = 0.96x
```

**GOD-дисконт (государство ~75% после конвертации):**
- Payout 25–50% → **-20%**
- Менеджмент = чиновники без KPI на капитализацию: **-5%**
- История допэмиссий (2015, 2025, конвертация 2026): **-5%**
- Итого GOD: **-30%**

```
Скорректированный P/BV = 0.96 × (1 − 0.30) = 0.67x
```

**Целевая цена (до конвертации):**
```
BV per share = 368 ₽
Целевая цена = 368 × 0.67 = 247 ₽
```

**Корректировка на конвертацию префов (базовый сценарий: разводнение ~35%):**
```
Акций после конвертации: 7 870 × 1.35 = ~10 625 млн шт.
BV per share после конвертации = (2 900 + 521) / 10 625 = ~322 ₽
Целевая цена = 322 × 0.67 = 216 ₽
```

**Средневзвешенная (40% без конвертации / 60% с конвертацией ~35%):**
```
my_fair_value = 247 × 0.40 + 216 × 0.60 = 99 + 130 = 229 ₽
Консервативно с дополнительной неопределённостью конвертации: 115 ₽
```

Консервативная оценка сохраняется на уровне **115 ₽** ввиду ключевой неопределённости параметров конвертации (возможный диапазон 61–229 ₽).

### Перекрёстная проверка

**Дивидендный метод (базовый):**
```
Прибыль FY 2025 = 500 млрд ₽
Payout 35% (середина диапазона 25–50%) = 175 млрд ₽
Кол-во акций (после конвертации) = ~10 625 млн шт.
DPS = 175 / 10 625 = ~16,5 ₽
Целевая дивдоходность = r − g = 20,5% − 5% = 15,5%
Целевая цена = 16,5 / 0.155 = ~106 ₽
```

Дивидендный метод даёт ~106 ₽ — подтверждает базовую оценку ~115 ₽.
