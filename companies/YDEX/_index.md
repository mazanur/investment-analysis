# Яндекс (YDEX)

> После реструктуризации 2024 года тикер изменился с YNDX на YDEX. Полный анализ под текущим тикером. Историческая карточка: [YNDX](../YNDX/_index.md).

## Бизнес-модель

Крупнейший технологический холдинг России. После реструктуризации 2024 года (выкуп иностранных акционеров с дисконтом ~50%) — полностью российская компания. Экосистема объединяет поиск, рекламу, e-commerce, такси, подписки, облако и финтех.

**Сегменты:**
| Сегмент | Описание | Доля выручки |
|---------|----------|--------------|
| Поиск и Портал | Контекстная реклама, Яндекс.Директ | ~45% |
| E-commerce | Яндекс Маркет, Лавка | ~25% |
| Райдтех | Такси, Доставка, Самокаты | ~20% |
| Плюс и развлечения | Подписка, Кинопоиск, Музыка | ~7% |
| Прочее | Облако, Образование, Финтех | ~3% |

**Драйверы роста:**
- **Реклама**: ~70% рынка интернет-рекламы совместно с VK. Рынок растет на 10-15% YoY
- **E-commerce**: Яндекс Маркет — 3-й игрок после Wildberries и Ozon. Убыточен, но растет на 40%+ YoY
- **Подписки**: Яндекс Плюс — 35+ млн подписчиков, ARPU растет
- **Райдтех**: лидер в такси и доставке, высокая маржинальность
- **AI/ML**: YandexGPT, Нейро-поиск — технологическое лидерство в РФ

## Конкурентные преимущества

1. **Монополия в поиске** — 65% поискового рынка РФ (Google ~30%), устойчивое доминирование
2. **Экосистема** — подписка Плюс связывает все сервисы, высокий retention и switching cost
3. **Технологии** — собственные ML-модели, YandexGPT, облачная инфраструктура
4. **Масштаб** — крупнейший IT-работодатель РФ (~25 тыс. сотрудников)
5. **Бренд** — узнаваемость и доверие на уровне Google в РФ

## Риски

1. **Регуляторное давление**: закон о маркетплейсах, маркировка рекламы (ЕРИР), ограничения контента
2. **Конкуренция с VK**: борьба за рекламные бюджеты, пересечение в соцсетях и контенте
3. **Отрицательный FCF**: инвестиции в e-commerce и облако съедают кэш (FCF LTM = -24.5 млрд)
4. **Концентрация на внутреннем рынке**: после реструктуризации — 100% РФ, нет международной диверсификации
5. **Высокие мультипликаторы**: P/BV 15x — дорого относительно балансовой стоимости, мало места для ошибки

## Оценка

### Расчет целевой цены по P/E

**Шаг 1: Базовые метрики**
- Чистая прибыль (LTM): 119.4 млрд руб
- Кол-во акций: 390.5 млн
- EPS = 305.8 руб
- Текущая цена: 4,759 руб

**Шаг 2: Целевой P/E**
- Ставка ЦБ (базовый сценарий, конец 2026): 13%
- ERP: 5%
- r = 18%
- g = 10% (высокий рост, ограничен потолком 12%)
- Спред r - g = 8% (>5%)

Базовый P/E = 1 / (0.18 - 0.10) = 12.5x

**Шаг 3: Поправка за качество**
| Фактор | Поправка |
|--------|----------|
| ROE 39% > сектора (20%) | +10% |
| Рост выручки 50% > сектора (15-20%) | +10% |
| Лидер сектора по масштабу | +10% |
| FCF отрицательный | -10% |
| Внутренний рынок, импортозамещение | +5% |
| **Итого** | **+25%** |

Целевой P/E = 12.5 x 1.25 = 15.6x

**Шаг 4: Целевая цена**
Целевая цена = 305.8 x 15.6 = 4,770 руб
Upside = (4,770 - 4,759) / 4,759 = 0%

### Перекрестная проверка: EV/EBITDA

- EBITDA (LTM): 242 млрд руб
- EV = Капитализация + Net Debt = 1,858 + 146 = 2,004 млрд руб
- EV/EBITDA = 8.3x

При целевом EV/EBITDA 8x (норма для растущего IT):
- Целевая EV = 242 x 8 = 1,936 млрд руб
- Целевая капитализация = 1,936 - 146 = 1,790 млрд руб
- Целевая цена = 1,790 / 390.5 = 4,584 руб

Результаты P/E и EV/EBITDA близки (4,770 vs 4,584) — оценка устойчива.

## Инвестиционный тезис

Яндекс — качественный актив с высоким ROE (39%), растущей маржой EBITDA (14% -> 21%) и монопольным положением в поиске (65% рынка). Экосистема (Плюс, Маркет, Такси, Облако) создает switching cost и рекуррентную выручку. Однако при текущей цене ~4,760 руб компания **справедливо оценена** — upside в базовом сценарии близок к нулю.

**Почему watch, а не buy:**
1. Нет запаса безопасности (margin of safety) — цена = fair value
2. Отрицательный FCF из-за инвестиций в e-commerce и облако
3. Высокий P/BV (15x) оставляет мало места для ошибки

**Когда покупать:**
- При коррекции до 3,500-4,000 руб (upside 20-35%)
- При ускорении снижения ставки ЦБ (до 10% вместо 13%)
- При выходе FCF в плюс (подтверждение качества прибыли)

**Катализаторы:**
1. Снижение ставки ЦБ — переоценка всех tech-компаний
2. Квартальные результаты с ростом маржи EBITDA >20%
3. Повышение дивидендов или формализация дивидендной политики
4. Выход e-commerce в безубыточность
