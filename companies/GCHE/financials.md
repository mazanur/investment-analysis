# Финансовые данные: Название (TICKER)

**Данные загружаются автоматически** в Investment API через job endpoints:

```bash
curl -X POST "$API_URL/jobs/fetch-smartlab/GCHE" -H "X-API-Key: $API_KEY"
```

Данные доступны через:
- `GET /companies/GCHE/reports?period_type=yearly` — годовые МСФО
- `GET /companies/GCHE/reports?period_type=quarterly` — квартальные МСФО

Claude при анализе читает данные из API, а не ходит на smart-lab каждый раз.

## Пометки о разовых статьях

Разовые / неповторяющиеся статьи, которые искажают показатели. Claude учитывает при расчёте нормализованной прибыли.

| Год | Статья | Сумма, млрд ₽ | Влияние на ЧП | Комментарий |
|-----|--------|---------------|---------------|-------------|
| | | | +/- X млрд | |

**Примеры разовых статей:**
- Списание гудвилла / обесценение активов
- Прибыль/убыток от продажи актива
- Курсовые переоценки (если нетипичные)
- Резервы под судебные иски
- Доход от переоценки инвестиций
