# Яндекс (YDEX)

> После реструктуризации 2024 года тикер изменился с YNDX на YDEX.

## Конкурентные преимущества

1. **Монополия в поиске** — 69.3% поискового рынка РФ, 67% на мобильных (+3.6 п.п.)
2. **Экосистема** — 110 млн активных пользователей, подписка Плюс связывает все сервисы
3. **Технологии** — YandexGPT, Алиса AI (19 млн пользователей), собственная облачная инфраструктура
4. **Масштаб** — ~31.5 тыс. основного персонала, крупнейший IT-работодатель РФ
5. **Бренд** — узнаваемость и доверие на уровне Google в РФ

## Риски

1. **Регуляторное давление**: закон о маркетплейсах, маркировка рекламы, ограничения контента
2. **Конкуренция с VK**: борьба за рекламный бюджет, пересечение в соцсетях
3. **Замедление рекламного рынка**: рост рекламной выручки замедлился до +13.5% (vs +20%+ ранее)
4. **Концентрация на РФ**: после реструктуризации — 100% внутренний рынок
5. **Высокие мультипликаторы**: P/BV ~11x — дорого относительно балансовой стоимости

## Оценка

### Расчёт целевой цены по P/E

**Шаг 1: Базовые метрики**
- Скорр. чистая прибыль (2025): 141.4 млрд ₽
- Кол-во акций: ~394 млн
- EPS = 359 ₽
- Текущая цена: 4,847 ₽

**Шаг 2: Целевой P/E**
- Ставка ЦБ (базовый сценарий, конец 2026): 13%
- ERP: 5%
- r = 18%
- g = 10% (высокий рост, ограничен потолком 12%)
- Спред r − g = 8% ✓

```
Базовый P/E = 1 / (0.18 − 0.10) = 12.5x
```

**Шаг 3: Поправка за качество**
| Фактор | Поправка |
|--------|----------|
| ROE ~55% >> сектора (20%) | +10% |
| Рост выручки 32% > сектора (15-20%) | +10% |
| Лидер сектора по масштабу | +10% |
| FCF улучшается, но не подтверждён | -5% |
| Внутренний рынок, импортозамещение | +5% |
| **Итого** | **+30%** |

```
Целевой P/E = 12.5 × 1.30 = 16.3x
```

**Шаг 4: Целевая цена**
```
Целевая цена = 359 × 16.3 = 5,852 ₽ ≈ 5,850 ₽
Upside = (5,850 − 4,847) / 4,847 = 21%
```

> Консервативная оценка с поправкой +25% (если считать FCF-риск на -10%): целевая 5,600 ₽, upside 16%.

### Перекрёстная проверка: EV/EBITDA

- EBITDA (скорр., 2025): 280.8 млрд ₽
- EV = Капитализация + Net Debt = 1,909 + 56 = 1,965 млрд ₽
- EV/EBITDA = 7.0x

При целевом EV/EBITDA 8x (норма для растущего IT):
- Целевая EV = 280.8 × 8 = 2,246 млрд ₽
- Целевая капитализация = 2,246 − 56 = 2,190 млрд ₽
- Целевая цена = 2,190 / 394 = 5,558 ₽ ≈ 5,560 ₽

Результаты P/E и EV/EBITDA близки (5,850 vs 5,560) — оценка устойчива. Среднее: ~5,700 ₽.

### Форвардная оценка (2026E)

При прогнозе менеджмента (EBITDA ~350 млрд) и допущении скорр. ЧП ~175 млрд:
- Forward EPS = 175 / 394 = 444 ₽
- Forward P/E = 4,847 / 444 = 10.9x ← дёшево для растущего tech

При целевом P/E 16.3x на 2026E: 444 × 16.3 = 7,237 ₽ (upside +49%).

## Инвестиционный тезис

Яндекс — качественный актив с рекордными результатами: скорр. ЧП +40% г/г, маржа EBITDA 19.5%, ROE ~55%, практически нулевой долг. Компания перешла от стадии инвестиций к зрелому генерированию прибыли. Начало дивидендных выплат (110 ₽, доходность 2.3%) и гайденс на 2026 (+20% выручки, EBITDA 350 млрд) подтверждают позитивный тренд.

**Почему watch, а не buy:**
1. Trailing upside 21% не дотягивает до порога 30%
2. FCF не раскрыт — качество прибыли не подтверждено
3. P/BV ~11x оставляет мало места для ошибки

**Почему sentiment bullish (повышен с neutral):**
1. Рекордные финансовые результаты по всем метрикам
2. Мультипликаторы сжались (P/E 15.6 → 13.5, EV/EBITDA 8.3 → 7.0) — рост прибыли опередил рост цены
3. Цикл снижения ставки ЦБ ускоряется (15.5%, тренд вниз)
4. Forward P/E 10.9x — дёшево для tech-лидера

**Когда покупать:**
- При коррекции до 4,000-4,500 ₽ (upside 30-46%)
- При ускорении снижения ставки ЦБ (до 10-12% вместо 13%)
- При подтверждении положительного FCF
- На форвардных мультипликаторах уже близко к зоне покупки

**Катализаторы:**
1. Снижение ставки ЦБ — переоценка всех tech-компаний
2. Квартальные результаты 2026 с подтверждением гайденса
3. Рост дивидендов (прогноз 135-155 ₽ за 2026)
4. Раскрытие FCF — подтверждение качества прибыли
