# Корпоративное управление: Сбербанк (SBER)

Данные для расчёта GOD-дисконта и оценки рисков корпоративного управления.
Автоматические секции обновляются: `make download-governance TICKER=SBER && make fill-governance TICKER=SBER`.

## Структура акционеров

| Акционер | Доля, % | Тип |
|----------|---------|-----|
| ЦБ РФ (через НКЦКИ→Минфин→ФНБ) | 50,0 + 1 акция | Государство |
| Free-float | 48,0 | Free-float |
| Казначейские | ~2,0 | Казначейские |

**Доля государства (прямая + косвенная):** 50%+1 акция (контролирующий пакет через ФНБ/Минфин)
**Казначейские акции:** ~2% (программа мотивации менеджмента)

## Дивидендная политика

**Текст политики:** Не менее 50% чистой прибыли по МСФО.

> "Сбербанк стремится направлять на выплату дивидендов не менее 50% чистой прибыли Группы по МСФО"

| Параметр | Значение |
|----------|----------|
| Payout ratio (цель) | 50% от ЧП МСФО |
| Периодичность | 1 раз в год |
| Дата ГОСА (обычно) | апрель-май |
| Реестр (обычно) | через 10-20 дней после ГОСА |

Политика стабильная, подтверждена менеджментом и Минфином.

## Расчёт GOD-дисконта

| Фактор | Значение | Дисконт |
|--------|----------|---------|
| Гос. владение > 50%? | да (50%+1) | применяется |
| Payout | 50% | базовый: **-10%** |
| Казначейские > 20%? | нет (~2%) | 0% |
| Рост дивидендов > 3 лет? | да (16→18,7→25→33,3→34,8) | **+5%** |
| Менеджмент-чиновники без KPI? | нет (Греф — бывший министр, но KPI привязаны к капитализации) | **0%** |
| **Итого GOD** | | **-5%** |

Примечание: в `_index.md` применён дополнительный санкционный дисконт ~35% сверх GOD из-за полных блокирующих санкций. Фактор «менеджмент-чиновники» не применяется, т.к. KPI менеджмента привязаны к капитализации (критерий RESEARCH_GUIDE: «чиновники **без** KPI на капитализацию»).
