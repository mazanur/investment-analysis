# Финансовые данные: Название (TICKER)

**Данные загружаются автоматически** в `data/` через скрипт:

```bash
make download TICKER=TICKER          # скачать для одной компании
make download                        # скачать для всех
```

Файлы сохраняются в:
- `data/smartlab_yearly.csv` — годовые МСФО
- `data/smartlab_quarterly.csv` — квартальные МСФО

Claude при анализе читает CSV из `data/`, а не ходит на smart-lab каждый раз.

## Когда нужно ручное вмешательство

Заполняй вручную **только** если:
1. Smart-lab не содержит данных по компании (новый тикер, недавнее IPO)
2. Данные на smart-lab устарели (последний период > 2 кварталов назад)
3. Нужны данные из РСБУ (smart-lab отдаёт только МСФО)

В этом случае — бери данные с e-disclosure.ru или dohod.ru и заполняй таблицы ниже.
