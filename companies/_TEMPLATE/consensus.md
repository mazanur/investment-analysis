# Консенсус-прогнозы аналитиков: Название (TICKER)

Данные брокерских аналитиков, недоступные Claude (за пейволлом / авторизацией).
Нужны для forward-оценки и перекрёстной проверки.

## Как заполнить

Источники (нужна авторизация клиента):
- **T-Investments**: приложение → карточка акции → «Прогнозы аналитиков»
- **BCS**: приложение или bcs-express.ru → аналитика → таргеты
- **SberCIB**: sberbank.com/sberCIB или приложение СберИнвестиции
- **Финам**: finam.ru → карточка акции → консенсус
- **Альфа-Банк**: приложение Альфа-Инвестиции

## Примечания

- Укажи, если таргеты устарели (> 3 мес.)
- Если аналитик пересмотрел таргет — запиши новый и старый
- Если есть расхождение > 30% между аналитиками — отметь причину
