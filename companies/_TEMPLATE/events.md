# Корпоративные события: Название (TICKER)

IR-материалы, пресс-релизы и предстоящие катализаторы.

## Как обновить

### Автоматически (MOEX ISS API)

Таблицы «Последние события» и «Предстоящие катализаторы» заполняются автоматически:

```bash
# 1. Скачать данные о событиях и дивидендах с MOEX
make download-events TICKER=TICKER

# 2. Сгенерировать events.md
make fill-events TICKER=TICKER

# Или для всех компаний сразу:
make download-events && make fill-events
```

Скрипт скачивает:
- **Дивиденды** — даты отсечки, размер выплаты
- **IR-календарь** — публикации отчётности (РПБУ, МСФО), конференц-звонки, ГОСА
- **Заседания ЦБ** — из `russia/macro.md`

Данные сохраняются в `data/moex_events.json`.

### Вручную (секции ниже)

Две секции требуют ручного заполнения — они **не перезаписываются** при повторном запуске:
- **Guidance менеджмента** — из конференц-звонков и IR-презентаций
- **Ключевые выдержки из IR-презентаций** — главные тезисы

Санкционный статус — см. [governance.md](governance.md).

Источники для ручного заполнения:
1. IR-страница компании → раздел «Пресс-релизы» / «Новости»
2. e-disclosure.ru → существенные факты
3. Поиск: "{тикер} отчётность 2026", "{тикер} дивиденды 2026"
