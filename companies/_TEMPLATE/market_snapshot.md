# Рыночные данные: Название (TICKER)

Smart-lab отдаёт цену, капитализацию, free-float и число акций автоматически.
Здесь — только то, чего **нет** на smart-lab.

## Как заполнить

1. ADV: https://www.moex.com/ru/issue.aspx?code=TICKER → «Итоги торгов»
2. Спред: торговый терминал (стакан заявок)
3. Казначейские акции: IR-страница компании или годовой отчёт
4. Привилегированные акции: moex.com (если есть TICKERP)

## Снимок

```yaml
snapshot_date: 2026-01-01

# Ликвидность (нет на smart-lab)
adv_30d_mln: 0               # млн ₽, среднедневной объём за 30 дней
bid_ask_spread_pct: 0         # %, типичный спред (из стакана)

# Структура капитала (частично есть на smart-lab)
shares_preferred_mln: 0       # млн шт, привилегированные (0 если нет)
treasury_shares_pct: 0         # %, казначейские акции (на балансе компании)

# Индексы (опционально)
moex_index_weight: 0           # %, вес в индексе МосБиржи (0 если не входит)
```
