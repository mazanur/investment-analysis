Т-Технологии (TCSG)

Бизнес-модель

Крупнейший цифровой банк России и финтех-экосистема. Основные направления:

  • Розничный банкинг — кредитные карты (исторически основной продукт), депозиты, накопительные счета, потребительские кредиты
  • Брокерский бизнес — Т-Инвестиции: 52,8 млн клиентов, 6,1 трлн ₽ активов под управлением
  • Страхование — Т-Страхование (включая бывший Росбанк Страхование)
  • Мобильный оператор — Т-Мобайл
  • Лайфстайл-сервисы — рестораны, путешествия, подписки

После интеграции Росбанка (1 января 2025) группа существенно выросла: активы 5,7 трлн ₽, кредитный портфель 3,1 трлн ₽.

Конкурентные преимущества

  • Цифровая модель — самый низкий cost-to-income среди крупных банков РФ, отсутствие физической сети (минимум отделений)
  • Экосистема — глубокая интеграция финансовых и лайфстайл-сервисов, высокий ARPU и retention
  • Бренд — один из самых узнаваемых финтех-брендов в России
  • Технологическое лидерство — собственная IT-платформа, высокая скорость разработки продуктов
  • Интеграция Росбанка — синергии ~20 млрд ₽/год, расширение корпоративного сегмента

Риски

Макро-риски

  • Высокая ставка ЦБ сжимает NIM (чистую процентную маржу)
  • Рост стоимости риска при экономическом замедлении

Операционные риски

  • Интеграция Росбанка — риски исполнения, временное давление на ROE
  • Высокая конкуренция с Сбером в брокерском бизнесе

Регуляторные риски

  • Ужесточение макропруденциальных требований (МПЛ)
  • Риск ограничений на комиссии (как в ЕС по interchange)

Специфические риски

  • Зависимость от IT-инфраструктуры — кибербезопасность критична
  • Высокий P/BV (1.4x) vs сектор — переоценка возможна при разочаровании

Мой тезис

TCSG — лучшая growth-история в российском финансовом секторе. Компания сочетает:

  1. Высокий ROE (29%) при умеренном P/BV (1.4x)
  2. Рост прибыли +40% YoY с прогнозом сохранения темпов
  3. Интеграцию Росбанка с синергиями ~20 млрд ₽/год
  4. Байбэк программу (до 5% капитала)
  5. Снижение ставки ЦБ как катализатор переоценки всего сектора

При текущем P/E 5.8x для компании с ROE 30% — это дёшево. Справедливый P/E для такого качества — 10-12x.

Position: BUY — значительный upside при понятных катализаторах.

Чеклист BUY

  • Upside > 30% — 65% в базовом сценарии, 42% в пессимистичном
  • Катализатор есть — снижение ставки ЦБ, интеграция Росбанка, байбэк
  • Тезис записан — growth + quality: ROE 30% при P/E 5.8x
  • Net Debt/EBITDA < 2.5x — не применимо для банка
  • Маржа стабильна — ROE восстановился с 24.6% (Q1) до 29.2% (Q3)
  • Ликвидность достаточна — ADV ~500 млн ₽, входит в индекс IMOEX
  • Качество прибыли — банк, OCF/NI не применимо; ROE стабилен
  • Геополитика учтена — не под SDN санкциями, внутренний рынок
  • Нет value trap признаков — прибыль растёт +40% YoY, ROE 30%

Результат: 9/9 — все критерии выполнены, position = BUY
