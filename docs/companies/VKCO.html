VK (VKCO)

Бизнес-модель

Крупнейшая социальная платформа России. После продажи игрового бизнеса My.Games (2024) сфокусирована на соцсетях, рекламе, EdTech и B2B-технологиях. Контролируется через «Газпром» и структуры, близкие к государству.

Сегменты:

Соцсети и медиаВКонтакте, Одноклассники, VK Видео, Дзен~71%
VK TechB2B-технологии, облако, почта для бизнеса~9%

Аудитория:

  • DAU: 78 млн пользователей
  • Охват российского интернета: >95%
  • Время на платформах: 51 млрд минут/день (+17% YoY)

Драйверы выручки:

  • Онлайн-реклама: 96.1 млрд (2024), +20% YoY — основной источник
  • Видеореклама: +69% YoY за 9М 2025 (3.7 млрд за 9М)
  • SME-реклама: +12% YoY за 9М 2025 (26.6 млрд за 9М)
  • VK Tech: +39% YoY за 9М 2025 — самый быстрорастущий сегмент

Конкурентные преимущества

  1. Монополия в соцсетях — ВКонтакте и Одноклассники не имеют прямых конкурентов в РФ после блокировки западных соцсетей
  2. Рекламный инвентарь — совместно с Яндексом занимает ~70% рынка интернет-рекламы РФ
  3. Аудитория — 78 млн DAU, охват >95% российского интернета — уникальный масштаб
  4. Экосистема — VK ID (авторизация), VK Pay, облако — повышают switching cost для бизнеса
  5. Видео-платформа — VK Видео занимает нишу после блокировки YouTube, рост вовлечённости

Риски

  1. Критический долг: Net Debt/EBITDA = 12.3x (LTM). Даже при EBITDA-прогнозе 20 млрд на 2025 год — это 3.9x. При ставке ЦБ 16% процентные расходы огромны
  2. Хронические убытки: чистый убыток каждый год (2020-2025), последний год -83.6 млрд (LTM). ROE = -58%. Бизнес разрушает акционерную стоимость
  3. Массивное размытие: допэмиссия 115 млрд руб привела к утроению числа акций (с 226 до 584 млн). EPS катастрофически размыт
  4. Отрицательный FCF: -34.4 млрд (LTM), отрицательный 4 года подряд. Бизнес не генерирует денежный поток
  5. Замедление роста: выручка замедлилась с +23% (2024) до +10% (9М 2025 YoY). Высокая ставка ЦБ давит на рекламные бюджеты
  6. Списания My.Games: резерв на 46.65 млрд по неполученным средствам от продажи игрового бизнеса
  7. Квази-государственное управление: контролируется через структуры Газпрома — менеджмент без KPI на капитализацию, непрозрачные решения
  8. Нет дивидендов: при отрицательной прибыли и огромном долге дивиденды невозможны на горизонте 3-5 лет

Дивиденды

VK никогда не платила дивиденды. При хронических убытках и критическом долге дивиденды невозможны. Горизонт потенциальных выплат — не ранее 2028-2029, при условии выхода на устойчивую прибыльность и снижения долга.

Оценка

Выбор метода

Net Debt/EBITDA = 12.3x — критический уровень (>3.5x). P/E не применим. Основной метод — EV/EBITDA + стресс-тест.

Компания убыточна, поэтому дополнительно используем EV/Sales для перекрёстной проверки.

Расчёт целевой цены по EV/EBITDA

Шаг 1: Базовые метрики (прогноз на 2025)

  • EBITDA прогноз 2025: 20 млрд руб (guidance менеджмента)
  • Net Debt (LTM): 77 млрд руб
  • Число акций: 584.4 млн
  • Текущая цена: 303 руб
  • Текущая капитализация: 177 млрд руб
  • Текущая EV = 177 + 77 = 254 млрд руб

Шаг 2: Целевой EV/EBITDA

  • WACC расчёт:
  • r_equity = 16% (ставка ЦБ) + 5% (ERP) = 21%
  • r_debt = 8.2% (средневзвешенная ставка VK, льготная)
  • D/(D+E): при EV = 254, Debt ~114 -> D/EV = 45%
  • E/EV = 55%
  • WACC = 21% x 0.55 + 8.2% x (1-0.22) x 0.45 = 11.6% + 2.9% = 14.5%
  • g = 3% (зрелый бизнес, рост замедлился до 10%, маржа неустойчива, долгосрочно — инфляционный рост)
  • tax = 22%

Целевой EV/EBITDA = (1 - 0.22) / (0.145 - 0.03) = 0.78 / 0.115 = 6.8x

Шаг 3: Поправка за качество

ФакторПоправка
Лидер в соцсетях, масштаб аудитории+10%
ROE -58% < сектора-10%
Маржа EBITDA нестабильна, падала-10%
Критический долг (12.3x)P/E не применим, EV/EBITDA основной
Зависимость от рекламных бюджетов-5%
Размытие акционеров (допэмиссия)-10%
Итого-25%

Целевой EV/EBITDA = 6.8 x 0.75 = 5.1x

Шаг 4: Целевая цена

  • Целевая EV = 20 x 5.1 = 102 млрд руб
  • Целевая капитализация = 102 - 77 = 25 млрд руб
  • Целевая цена = 25 / 584.4 = 43 руб

Этот результат показывает, что при текущем долге equity value близка к нулю.

Оптимистичный сценарий (EBITDA = 30 млрд, ставка 13%)

Если VK достигнет EBITDA 30 млрд (2026E) и ставка снизится до 13%:

  • r_equity = 18%, WACC = 12.8%
  • Целевой EV/EBITDA = 0.78 / (0.128 - 0.03) = 8.0x
  • С поправкой -25%: 8.0 x 0.75 = 6.0x
  • Целевая EV = 30 x 6.0 = 180 млрд
  • Целевая капитализация = 180 - 77 = 103 млрд
  • Целевая цена = 103 / 584.4 = 176 руб
  • Upside: -42%

Стресс-тест (обязателен при >3.5x)

Процентные расходы при ставке ЦБ +3п.п. (19%):

  • Долг: 114.7 млрд руб
  • Средневзвешенная ставка VK: 8.2% (льготная, частично субсидированная)
  • При рыночной ставке 19% + 2% спред = 21%: процентные расходы = 114.7 x 0.21 = 24.1 млрд руб
  • EBITDA прогноз: 20 млрд руб
  • Проценты / EBITDA = 120% — процентные расходы ПРЕВЫШАЮТ EBITDA

Красный флаг: при рыночных ставках компания не способна обслуживать долг из операционного денежного потока. Выживание зависит от льготного финансирования и допэмиссий.

Перекрёстная проверка: EV/Sales

  • Выручка LTM: 153 млрд руб
  • Текущий EV/Sales = 254 / 153 = 1.7x
  • Для убыточной tech-компании с маржой EBITDA 4%: целевой EV/Sales = 0.5-1.0x (бенчмарк — убыточные аналоги)
  • При EV/Sales = 0.7x: целевая EV = 107 млрд, капитализация = 107 - 77 = 30 млрд, цена = 51 руб
  • При EV/Sales = 1.0x: целевая EV = 153 млрд, капитализация = 153 - 77 = 76 млрд, цена = 130 руб

Перекрёстная проверка подтверждает: справедливая цена значительно ниже текущей.

Инвестиционный тезис

VK — структурно убыточная компания с критическим долгом и разрушенной акционерной стоимостью. При Net Debt/EBITDA 12.3x и хронических убытках (5 лет подряд) equity value близка к нулю. Допэмиссия на 115 млрд руб утроила количество акций, размыв существующих акционеров. Стресс-тест показывает, что при рыночных ставках процентные расходы превышают EBITDA — компания зависит от льготного финансирования.

Почему avoid:

  1. Net Debt/EBITDA = 12.3x — критический уровень, стресс-тест провален
  2. Чистый убыток 5 лет подряд, ROE = -58%, FCF отрицательный 4 года
  3. Допэмиссия утроила акции — массивное размытие, сигнал финансового бедствия
  4. Даже в оптимистичном сценарии (ставка 10%, EBITDA 30 млрд) upside = 0%
  5. Нет катализатора для переоценки вверх в обозримом горизонте

Что может изменить тезис (маловероятно):

  • Выход на устойчивую EBITDA >35 млрд (x2 от прогноза) + снижение долга ниже 50 млрд
  • Структурная реорганизация (spin-off VK Tech, IPO отдельных сегментов)
  • Агрессивное снижение ставки ЦБ до <10% в сочетании с ростом маржи

Чеклист

  • Upside > 30% — НЕТ (-49% в базовом сценарии)
  • Катализатор есть — НЕТ (нет видимого катализатора)
  • Тезис записан — да
  • Net Debt/EBITDA < 2.5x — НЕТ (12.3x, критический)
  • Маржа стабильна или растёт — НЕТ (рухнула с 23% до 4%)
  • Ликвидность — высокая, голубая фишка
  • OCF/Net Income > 0.5 — НЕТ (убыточна, FCF отрицательный)
  • Геополитика учтена — внутренний рынок

Результат: 3/8 — position = avoid.
