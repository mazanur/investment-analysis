Башнефть привилегированные (BANEP)

Бизнес-модель

Вертикально-интегрированная нефтяная компания, дочерняя структура Роснефти:

  • Добыча нефти — месторождения в Башкортостане и ХМАО
  • Переработка — НПЗ «Башнефть-Уфанефтехим», «Башнефть-Новойл», «Башнефть-УНПЗ» (мощность ~20 млн тонн/год)
  • Сбыт — розничная сеть АЗС «Башнефть»

Ключевая особенность: преференции — дивиденды по привилегированным акциям имеют приоритет, одинаковый размер с обыкновенными при значительно меньшей цене.

Конкурентные преимущества

  • Нулевой долг — Net Debt = 0, финансовая крепость
  • Высокая дивидендная доходность — преференции торгуются с дисконтом к обычкам
  • Стабильный денежный поток — нефтепереработка менее волатильна чем добыча
  • Часть Роснефти — доступ к ресурсам и синергиям

Риски

Макро-риски

  • Падение цен на нефть снижает маржу переработки
  • Укрепление рубля негативно для экспортной выручки

Операционные риски

  • Зрелые месторождения, падение добычи
  • Капексы на модернизацию НПЗ

Регуляторные риски

  • Налоговая нагрузка (НДПИ, демпфер)
  • Санкции на нефтепродукты

Корпоративные риски

  • Миноритарный статус — зависимость от решений Роснефти
  • Низкая ликвидность преференций

Мой тезис

BANEP — одна из самых недооценённых историй на рынке:

  1. P/E 1.6x, P/BV 0.2x — экстремально низкие мультипликаторы
  2. Нулевой долг — компания не страдает от высокой ставки ЦБ
  3. Дивидендная доходность 11-16% — стабильные выплаты
  4. Дисконт преф к ао ~50% — при одинаковых дивидендах

Причины дисконта:

  • Миноритарный статус (Роснефть контролирует)
  • Низкая ликвидность
  • Отсутствие права голоса (но для дивидендного инвестора неважно)

Position: BUY — экстремальная недооценка с высокими дивидендами.
