# Пересмотр всех компаний с position: buy по RESEARCH_GUIDE.md

Полный пересмотр инвестиционных карточек 12 компаний с текущим position: buy. Для каждой компании заново выполняется полный цикл анализа по RESEARCH_GUIDE.md: сбор свежих финансовых данных со smart-lab, пересчёт справедливой цены с актуальной ставкой ЦБ (16%), обновление тезиса и position.
Эта задача уже была запущена, некоторые задачи уже были сделаны, ориентируйся на чекбоксы

## Контекст

- **Ставка ЦБ**: 16,00% (снижена 19.12.2025)
- **Прогноз ставки**: 12-13% к концу 2026
- **Инфляция**: 5,59% (2025), всплеск в январе 2026 из-за НДС 22%
- **ERP**: 5% (стандарт для российского рынка)
- Files involved:
  - `companies/{TICKER}/_index.md` — 12 файлов (основной объект работы)
  - `companies/{TICKER}/trend.json` — обновляется скриптом после пересмотра
  - `companies/_index.md` — сводная таблица компаний
  - `_index.md` — таблица статуса обновлений
  - `russia/macro.md` — макро-контекст (ставка, инфляция)
  - `sectors/{sector}/_index.md` — секторальный контекст
  - `scripts/generate_trend_json.py` — скрипт генерации trend.json
- Related patterns: существующий формат карточек (YAML frontmatter + markdown секции)
- Dependencies: smart-lab.ru (финансовые данные), moex.com (цены, объёмы)

## Стратегия

- Обрабатывать компании по секторам — чтобы секторальный контекст читался один раз
- Для каждой компании: WebFetch smart-lab (годовые + квартальные МСФО) → пересчёт → обновление карточки
- Формула оценки: P/E = 1/(r-g) для компаний с низким долгом, EV/EBITDA для закредитованных, P/BV для банков
- После обновления каждой компании — обновить trend.json скриптом
- В конце — обновить сводные таблицы и закоммитить

## Для каждой компании выполняется (чеклист из RESEARCH_GUIDE)

1. Прочитать секторальный контекст
2. Прочитать макро (ставка 16%, прогноз 12-13%)
3. WebFetch smart-lab: годовые МСФО + квартальные МСФО
4. Обновить финансовые таблицы в карточке
5. Понять бизнес-модель (обновить если изменилась)
6. Оценить финансовое здоровье (красные/зелёные флаги)
7. Проверить stop-условия
8. Рассчитать справедливую цену (выбрать метод по долгу):
   - P/E = 1/(r-g) x поправка — для компаний с Net Debt/EBITDA < 2.5x
   - EV/EBITDA = (1-tax)/(WACC-g) x поправка — для закредитованных
   - P/BV = ROE/(r-g) — для банков и страховых
   - NAV — для холдингов
9. Три сценария ставки (пессимистичный/базовый/оптимистичный)
10. Чувствительность к g
11. Перекрёстная проверка (DCF + дивидендный)
12. GOD-дисконт если госкомпания
13. Определить upside и position (buy >30%, watch 15-30%, hold 0-15%, sell <0%)
14. Обновить тезис
15. Обновить YAML-метаданные
