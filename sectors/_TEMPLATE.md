## Инструкция по заполнению

1. **YAML-метаданные**: обязательны для быстрой фильтрации
2. **Обзор**: контекст для понимания специфики отрасли
3. **Метрики**: только проверенные цифры с источниками
4. **Драйверы/Риски**: 3-5 пунктов, ранжированных по значимости
5. **Регуляторика**: важна для госсектора и экспорта
6. **Макро-влияние**: как ставка/курс/инфляция бьют по отрасли
7. **Компании**: таблица с быстрым sentiment-чеком
8. **Источники**: обязательны для верификации

**Обновление**: при изменении ключевых метрик обновляй поле `updated` в YAML.
