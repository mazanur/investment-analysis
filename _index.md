# Инвестиционный анализ

Иерархическая система документов для фундаментального анализа российских компаний.

## Структура

| Уровень | Описание | Путь |
|---------|----------|------|
| Глобальный | Мировая экономика | [global/](global/) |
| Россия | Макроэкономика РФ | [russia/](russia/) |
| Секторы | Отраслевой анализ | [sectors/](sectors/) |
| Компании | Карточки эмитентов | [companies/](companies/) |

## Как использовать

1. Заполняй документы сверху вниз (от глобального к компаниям)
2. При анализе конкретной компании читай снизу вверх
3. Обновляй поле `updated` при каждом изменении
4. Используй `sentiment` для быстрой оценки: bullish / neutral / bearish
